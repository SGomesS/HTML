Oque é html ?

Inicio

HTML, ou HyperText Markup Language, é a linguagem de marcação usada para criar páginas da web. É a base da estrutura de praticamente todas as páginas da web e é interpretado pelos navegadores da web para exibir o conteúdo na forma de texto, imagens, links, formulários e outros elementos interativos.



Oque são Tags?

Inicio

HTML funciona por meio de "tags" ou "etiquetas" que você insere no código para marcar elementos específicos em uma página da web. Cada tag tem uma função específica.

Tags Basicas

Todas as Tags são definidas da seguinte forma < Nome da TAG > .Segue listado abaixo algumas tags principais:

  • < html > : Define o início e o fim de um documento HTML.
  • < head >: Contém informações sobre o documento, como o título da página e metadados.
  • < title > : Define o título da página, que é exibido na aba do navegador.
  • < body > : Contém o conteúdo visível da página, como texto, imagens e links.
  • < h1 >, < h2 >, < h3 >, ... < h6 > : Criam cabeçalhos com diferentes níveis de importância.
  • < p > : Define um parágrafo de texto.
  • < a > : Cria um link para outra página ou recurso.
  • < img > : Inclui uma imagem na página.
  • < ul > : Cria uma lista não ordenada.
  • < ol > : Cria uma lista ordenada.
  • < dl > (Definição de Lista): Define o início e o fim da lista de glossários.
  • < dt > (Definição de Termo): Define o termo ou palavra a ser explicado.
  • < dd > (Definição de Descrição): Define a descrição ou explicação do termo.
  • < li > : Define um item de lista em < ul > ou < ol >.


Listas numeradas

Inicio

As listas numeradas em HTML, também conhecidas como "listas ordenadas", são um tipo de elemento HTML usado para criar listas em que cada item é numerado sequencialmente. Os números são automaticamente gerados pelo navegador, começando a partir de 1 por padrão. As listas numeradas são úteis para criar listas de itens que precisam ser organizados em uma ordem específica.



Listas não numeradas

Inicio

As listas não numeradas em HTML, também chamadas de "listas não ordenadas", são um tipo de elemento HTML usado para criar listas de itens que não são necessariamente organizados em uma ordem específica. Em vez de números, essas listas usam marcadores, como pontos, círculos ou quadrados, para indicar cada item.



Listas de Glossario

Inicio

Listas de glossários em HTML são usadas para apresentar termos ou palavras junto com suas respectivas definições ou explicações. Essas listas são comumente usadas em sites e documentos acadêmicos para ajudar os leitores a entender termos técnicos ou específicos de um determinado assunto.
